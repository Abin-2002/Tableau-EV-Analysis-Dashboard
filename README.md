# Electric Vehicle Data Analysis

## Project Overview
This project analyzes Electric Vehicle (EV) adoption trends using real-world data. The analysis focuses on understanding growth patterns across years, regions, manufacturers, and vehicle types to generate insights that support data-driven decisions in the EV industry.

## Objectives
- Analyze EV adoption trends over time  
- Compare Battery Electric Vehicles (BEV) and Plug-in Hybrid Electric Vehicles (PHEV)  
- Identify top-performing EV brands and models  
- Evaluate the impact of electric range on EV adoption  
- Generate insights for business, policy, and sustainability decisions  

## Dataset Information
- <a href = "https://github.com/Abin-2002/Tableau-EV-Analysis-Dashboard/blob/main/Dataset.csv"> Dataset </a>
- Source: Kaggle  
- Type: Electric Vehicle Population Data  
- Region: United States  

### Key Attributes
- Vehicle Make and Model  
- Model Year  
- Electric Vehicle Type (BEV / PHEV)  
- Electric Range  
- CAFV (Clean Alternative Fuel Vehicle) Eligibility  

## Methodology
- Data Cleaning and Preprocessing  
- Exploratory Data Analysis (EDA)  
- Data Aggregation using Pivot Tables  
- KPI Calculation and Performance Metrics  
- Data Visualization using Tableau  
- Dashboard Development for interactive analysis  

## Key Performance Indicators (KPIs)
- Total Number of Vehicles  
- Average Electric Range  
- Total BEV Vehicles and Percentage  
- Total PHEV Vehicles and Percentage  

## Data Visualization
- Line Charts for trend analysis  
- Bar Charts for comparison analysis  
- Donut Charts for distribution insights  
- Map Visualization for regional analysis  
- Tree Maps for hierarchical data representation

- 📊 **View Dashboard:**
- <a href = "https://github.com/Abin-2002/Tableau-EV-Analysis-Dashboard/blob/main/Dashboard_Preview%20(2).png"> Dashboard </a>

## Key Insights
- Tesla holds over 50% market share in EV adoption  
- EV registrations show consistent growth from 2012 to 2023  
- BEVs account for approximately 80% of total EVs  
- Average electric range is around 73 miles  

## Recommendations
- Increase competition among EV manufacturers  
- Promote adoption of Battery Electric Vehicles (BEVs)  
- Improve battery efficiency and vehicle range  
- Focus on regions with low EV adoption  

## Tools and Technologies
- Tableau (Data Visualization and Dashboarding)  
- Microsoft Excel (Data Processing)  
- Kaggle (Dataset Source)  

## Conclusion
The project demonstrates strong growth in the EV market, with a clear shift toward fully electric vehicles. The insights derived from this analysis can help stakeholders make informed decisions regarding EV adoption, infrastructure development, and sustainability strategies.
